    active(path) click to toggle source

    The active method is a helper function that returns “active” or empty string. It is used to set a CSS class to active for highlighting the active menu item.

    # File app/helpers/application_helper.rb, line 7
def active(path)
  if (path == home_index_path && request.fullpath == '/') ||
     (path == '/settings' && (request.fullpath =~ /^\/group/).present?) ||
     (path == '/settings' && (request.fullpath =~ /^\/project/).present?) ||
     (path == '/settings' && (request.fullpath =~ /^\/users\/edit/).present?) ||
      request.fullpath =~ /^#{path}/
    'active'
  end
end
    selection_options(selection_hash) click to toggle source

    The #selection_options method generates a selection list array based on a hash that is pasted as the single parameter. The hash will use the key as the label and value as the id.

    # File app/helpers/application_helper.rb, line 22
def selection_options(selection_hash)
  options = []
  selection_hash.each do |label, id|
    options << [label, id]
  end
  return options
end

    tag_labels(tags) click to toggle source

    The #tag_labels helper method will take an array of tag strings and return HTML code that wraps the elements in button CSS code.

    # File app/helpers/application_helper.rb, line 79
def tag_labels(tags)
  html = ''
  if tags.present?
    tags.each do |tag|
      html += "<span class='label'>" + tag + "</span>&nbsp;"
    end
  end
  html.html_safe
end
    tag_options_list(tagged_object=nil) click to toggle source

    The tag_options method returns an option_for_select object for all the tags currently in the system or associated with the object that is passed as the optional parameter.

    # File app/helpers/application_helper.rb, line 60
def tag_options_list(tagged_object=nil)
  tag_list = tag_options = []
  tag_list = Macrophage.all_tags + ImmuneResponse.all_tags
  tag_list.uniq!
  tag_list.sort!

  tag_list.each do |tag|
    tag_options << [tag, tag]
  end
  selected_tags = tagged_object.tags if tagged_object && tagged_object.tags.present?

  return options_for_select(tag_options, selected_tags)
end
    value_or_none(value) click to toggle source

    The #value_or_none helper method will print the value or a None string followed by a <br/> html code. The method returns html_safe string.

    # File app/helpers/application_helper.rb, line 94
def value_or_none(value)
  html = ''
  if value.present?
    html += value
  else
    html += 'None<br/>'
  end
  html.html_safe
end
